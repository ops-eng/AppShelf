AppShelf Help

Version: 1.0.0 (2)

What AppShelf Does

AppShelf is a lightweight app launcher for macOS that helps you open selected apps quickly from one organized grid.

Getting Started

  1. Open Settings from the menu bar icon menu.
  2. In Manage Apps, click Add App and choose applications.
  3. Reorder apps with drag and drop.
  4. Use the Pro tab to unlock advanced customization.

Main Features

  • Add, remove, and reorder apps in your shelf.
  • Launch apps from a clean visual grid.
  • Customize appearance (columns, icon size, spacing, hide delay) in Pro.
  • Customize global activation shortcut in Pro.
  • Restore purchases from the Pro tab.
  • Optional Launch at Login support.

Troubleshooting Shortcut

Changes take effect immediately.

  1. Check System Settings > Privacy & Security > Accessibility.
  2. Ensure AppShelf is listed and enabled.
  3. Check if another app already uses the same shortcut.
  4. Try a unique key combination for testing.
